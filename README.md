# [Easy Browser](https://github.com/oi-contrib/Easy-Browser)
一个简单的定制化浏览器

## 如何开发？

确定node.js已经安装好了以后，直接执行下面目录安装依赖：

```js
npm install
```

然后，执行下面的命令即可启动开发：

```js
npm run dev
```

开发完毕，需要打包发布，直接：

```js
npm run build
```

打包后的安装包位于 release 中。
